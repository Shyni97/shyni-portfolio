# Certificate Images

Please upload your certificate images to this directory with the following names:

1. **python-cert.png** - Python for Beginners certificate from University Of Moratuwa
2. **mongodb-cert.png** - MongoDB Skill - CRUD Operations certificate from MongoDB
3. **frontend-cert.png** - Front-End Web Development certificate from University Of Moratuwa

## Image Guidelines:
- Use PNG or JPG format
- Recommended size: 1200x800 pixels or similar aspect ratio
- Ensure certificates are clear and readable
- File names must match exactly as listed above

The flip cards will display these certificate images when clicked.
